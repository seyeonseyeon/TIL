# DAY05 (git hub특강)

## 1. GIT CLONE: 협업할 때

#### 1. 만들고 보내는 방법

1. git init
2. touch 파일이름
3. git add. 
4. git commit -m "업로드 한다."
5. git log --oneline: 확인
6. git remote add origin URL: 원격 저장소 연결
7. git push origin master


#### 2. 받는 방법

1. git clone URL
2. git pull origin master
3. git add.
4. git commit - m "업로드 한다."
5. git push origin master
